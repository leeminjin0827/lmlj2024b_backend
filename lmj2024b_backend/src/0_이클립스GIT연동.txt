[ 이클립스 GITHUB 연동 ]
	1. GITHUB 웹페이지에서 저장소(리포지토리) 만들기
	2. [ new repository ] 클릭
		1. Repository name : lmj2024b_backend
		2. Public(공개용) 체크
	3. 방금 생성된 비어 있는 저장소 확인 할 수 있다.
	4. [이클립스] -> 상단메뉴[window]->[show view]->[other]
	5. [git] -> [git repository]
	6. clone a git repository 클릭 -> clone URL 클릭
	7.
		URI : 클론할 깃 저장소 주소
		user : git 아이디
		password : git패스워드
	8. [ next ] -> [ finish ]
	9. 현재 이클립스내 프로젝트를 clone된 저장소에 공유하기
		1. 공유할 프로젝트폴더 오른쪽 클릭
		2. [ team ] share project
			* 주의할점 : 상단에 use create 체크 안함
			repository : 프로젝트를 공유할 저장소 선택
			[ finish ]
			* 프로젝트폴더명 옆에 [저장소명] 있으면 공유 성공
		
	10. [최초로 커밋/푸쉬]
		프로젝트명 오른쪽클릭 -> team -> commit
